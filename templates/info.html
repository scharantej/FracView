Mandelbrot Ensemble

The Mandelbrot ensemble is a set of all parameter values for a given complex quadratic equation that produce a bounded set of iterates.

The most famous member of the Mandelbrot ensemble is the Mandelbrot set, which is defined by the following equation:

z_{n+1} = z_n^2 + c

where z0 = 0 and c is a complex constant.

The Mandelbrot set is a fractal with a very complex and intricate structure. It is known for its self-similarity and its ability to generate beautiful and unexpected patterns.
